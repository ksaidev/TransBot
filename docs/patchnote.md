>

> * 링크, 이메일 유지
>

>
>     수동 모드에서 불편하게 남아있던 '1'표시, 이제는 수동 모드일 때에도 메시지가 자동 읽음처리되어 읽지 않음 표시가 남아있지 않습니다!
